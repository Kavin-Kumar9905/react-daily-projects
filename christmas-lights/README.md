# Christmas Lights

Christmas Lights is a React application that simulates a string of
animated lights, similar to decorative lights used during the
Christmas season.

This project is part of a **daily React practice series**, focusing on
time-based state changes and safe effect handling.

---

## Problem Statement

The application displays a row of colored circles whose brightness
changes over time, creating a rippling light effect.

The animation can be started, stopped, and controlled by the user.

---

## Features

- Displays 7 lights in a row
- One light is highlighted at a time
- Ripple animation effect using a timer
- Start / Stop control
- Adjustable interval to control animation speed

---

## Tech Stack

- React
- Vite
- JavaScript
- Bootstrap
