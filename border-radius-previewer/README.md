# Border Radius Previewer

Border Radius Previewer is a React application that allows users to visually
adjust the `border-radius` property and instantly preview the resulting shape.

This project is part of a **daily React practice series** focused on mastering
frontend fundamentals through small, focused projects.

---

## Problem Statement

The CSS `border-radius` property can take multiple values to control the shape
of an elementâ€™s corners. This tool helps users understand how these values work
by providing real-time visual feedback.

---

## Features

- Live preview of a box with adjustable border-radius
- Independent control of all 4 corners:
  - Top-left
  - Top-right
  - Bottom-right
  - Bottom-left
- Automatically generated CSS output
- One-click copy to clipboard
